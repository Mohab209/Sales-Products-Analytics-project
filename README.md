## Sales-Products-Analytics-project
In this project I Analyze Sales & Product Data To extract Revenue, Profit, and Orders Trends.
This dashboard provides a comprehensive analysis of sales performance across products, cities, and time periods. It blends geospatial, time series, and profitability insights to support strategic decision-making.

## Steps In this Project
1. **Import Data** from **Excel** to **Power BI**  
2. **Data Cleaning** with **Power Query**  
3. **Data Modeling** build relations between tables with **Power BI**  
4. **Data Analysis** analyze data and create measures  
5. **Data Visualization** with **Power BI**
